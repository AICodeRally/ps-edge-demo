# Engagements

Manage all client consulting engagements from planning through completion. Track status, budget, timeline, and deliverables.

## Stat Cards

- **Active Value** - Total contract value of active engagements
- **Active Count** - Planning + In Progress engagements
- **Completion Rate** - On-time delivery percentage
- **Avg Cycle Time** - Days from start to completion
- **Risk Count** - Engagements flagged for attention

## Engagement Types

- **STRATEGIC_PLANNING** - Long-term strategy development
- **CAPACITY_BUILDING** - Skill and capability enhancement
- **PROGRAM_EVALUATION** - Impact assessment and measurement
- **FUNDRAISING** - Development and donor strategy
- **BOARD_DEVELOPMENT** - Governance and leadership
- **DIGITAL_TRANSFORMATION** - Technology adoption

## Status Workflow

1. **PLANNING** - Scoping and resource allocation
2. **IN_PROGRESS** - Active delivery
3. **REVIEW** - Client review and feedback
4. **COMPLETED** - Deliverables accepted
5. **ON_HOLD** - Paused by client

## Quick Actions

- **Filter by status** - Use dropdown to view specific statuses
- **Click any row** - View full engagement details
- **+ New Engagement** - Create new client project

## Best Practices

- Update status weekly during team sync
- Flag engagements as "at risk" early (don't wait for deadline)
- Maintain 85%+ on-time completion rate
- Review blocked engagements daily
