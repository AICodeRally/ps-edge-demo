# Project Management

## What Are Projects?

- **New program launches** - Keys to Success expansion
- **System improvements** - Case management software
- **Facility upgrades** - Classroom renovations
- **Curriculum development** - New life skills modules

## Project Status Definitions

**On Track:**
- Meeting milestones
- Budget aligned with progress
- No blockers

**At Risk:**
- 1-2 weeks behind schedule
- Minor budget concerns
- Solvable blockers

**Behind:**
- >2 weeks behind
- Budget issues
- Major blockers

**Completed:**
- All milestones done
- Final report submitted
- Lessons learned documented

## Managing At-Risk Projects

**Immediate Actions:**
1. Identify root cause of delay
2. Update timeline or reduce scope
3. Reallocate resources if needed
4. Communicate with stakeholders
5. Document mitigation steps

## Best Practices

### Project Planning
- Define clear, measurable milestones
- Set realistic timelines
- Assign project lead
- Identify dependencies

### Weekly Check-ins
- Review milestone progress
- Update completion %
- Flag blockers early
- Adjust timeline if needed

### Financial Tracking
- Track spending weekly
- Compare to progress
- Request budget adjustments early
- Document all expenditures

### Stakeholder Communication
- Monthly status updates
- Proactive problem escalation
- Celebrate milestone completions
- Share lessons learned
