# Programs Overview

Manage all AFFCF programs serving Arizona foster youth.

## Active Programs (8)

### Education & Academic Support
- **Keys to Success** - After-school tutoring (142 students)
- **Scholarship Program** - College funding (34 recipients)

### Life Skills & Development
- **Life Skills Training** - Independent living skills (89 youth)
- **Career Readiness** - Job prep and internships

### Transition Support
- **Transition Services** - Ages 18-21 aging out (58 participants)
- **Housing Assistance** - Stable housing placement

### Youth Development
- **Activity Funding** - Sports, arts, enrichment
- **Mentoring Program** - One-on-one support

## Program Metrics to Track

**Enrollment:**
- Current enrollment vs capacity
- Waitlist size and management
- Eligibility verification status

**Outcomes:**
- Grade level improvement (target: 2+ levels)
- Attendance rates (target: 85%+)
- College enrollment (target: 60%+)
- Satisfaction scores (target: 90%+)

**Budget:**
- Spending vs budget by program
- Cost per beneficiary
- Funding sources (grants, donations, fees)

## Best Practices

### Program Enrollment
1. Verify foster care eligibility
2. Obtain parent/guardian consent
3. Complete intake assessment
4. Assign to appropriate program
5. Set individual goals

### Outcome Measurement
- **Track progress monthly**
- Use standardized assessments
- Document success stories
- Report to funders quarterly

### Budget Management
- Monitor spending weekly
- Flag programs over 90% budget
- Reallocate funds as needed
- Track restricted vs unrestricted
